# ③ ディープラーニング

> 計画 6/25（基礎）/ 6/28（手法）。Phase1（7月）の前哨。**勾配の流れ**と**各アーキの帰納バイアス**を軸に。

## 学習の基本機構
- **MLP**：アフィン変換＋非線形活性化の合成。単層パーセプトロンは線形分離限界（XOR不可）。
- **誤差逆伝播**：計算グラフ上での**連鎖律**による勾配計算。順伝播で活性を保持し、逆伝播で ∂L/∂w を出力側から伝播。
- **最適化**：ミニバッチSGD。系譜 = SGD → Momentum（慣性）→ AdaGrad（学習率の座標別減衰）→ RMSProp（指数移動平均）→ **Adam**（Momentum＋RMSProp）。学習率が支配的ハイパラ。

## 活性化関数と勾配消失
- **sigmoid/tanh** は飽和域で導関数→0、深層で勾配が指数的に減衰（**勾配消失**）。
- **ReLU**：正領域の勾配が1で消失を緩和、疎活性・高速。難点は **dying ReLU** → Leaky/Parametric ReLU, GELU 等。
- **softmax**：多クラス出力を確率単体へ。交差エントロピーと組で勾配が綺麗。
- 補助技術：**バッチ正規化**（内部共変量シフト緩和、勾配安定）、適切な初期化（Xavier/He）。

## 正則化
- **ドロップアウト**：学習時にユニットを確率的にマスク＝暗黙のアンサンブル。
- **重み減衰（L2）**、**早期終了**、**データ拡張**。

## 主要アーキテクチャ

| 種類 | 帰納バイアス | 要点 |
|---|---|---|
| **CNN** | 局所性・並進不変性 | **畳み込み＝重み共有＋局所受容野**、プーリングでダウンサンプリング |
| **RNN** | 時系列・パラメータ共有 | **BPTT**。長期依存で勾配消失/爆発 |
| **LSTM/GRU** | ゲート制御 | セル状態＋ゲートで長期依存を保持、勾配経路を確保 |
| **Transformer** | 全結合的な系列関係 | **自己注意**。逐次性を排し並列化 |
| **GAN** | 敵対的学習 | 生成器 vs 識別器のminimaxゲーム |
| **拡散モデル** | 反復ノイズ除去 | 前向き拡散の逆過程を学習（SD等） |

### CNN
畳み込みは局所受容野＋重み共有でパラメータ削減と並進不変性を獲得。プーリング（max/avg）で解像度圧縮と微小変位への頑健性。**ResNet** は残差接続 `y = F(x) + x` で勾配を恒等経路で流し、**劣化問題（degradation）**を解消して超深層化を可能に。

### Transformer / Attention
- **自己注意**：Q,K,V を線形射影し `softmax(QKᵀ/√d_k)V`。各位置が全位置を直接参照。
- **RNN比の利点**：逐次依存を排し**並列計算可**、任意距離の依存を一定パスで捕捉（長距離依存に強い）。代償は系列長 n に対し **O(n²)** の計算/メモリ。
- 位置情報は**位置エンコーディング**で注入。マルチヘッドで部分空間ごとに関係を学習。
- 事前学習＋ファインチューニング（BERT=エンコーダ/MLM、GPT=デコーダ/自己回帰）。**転移学習**の基盤。

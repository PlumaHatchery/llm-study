# ③ ディープラーニング

> 計画 6/15〜6/17 の範囲。**7月のLLM学習の先取り**にもなる重要パート。

## ニューラルネットワークの基礎
- **パーセプトロン** … 入力×重み＋バイアス → 活性化関数。単層では**XORが解けない**（線形分離不可）。
- **多層パーセプトロン（MLP）** … 隠れ層を持つ。**誤差逆伝播法（バックプロパゲーション）**で重みを更新。
- **ディープラーニング** … 隠れ層を多数重ねたNN。**特徴量を自動で獲得**できるのが革新点。

## 活性化関数

| 関数 | 特徴 |
|---|---|
| シグモイド | 0–1に圧縮。**勾配消失**しやすい |
| tanh | −1〜1。シグモイドより勾配が大きい |
| **ReLU** | 正はそのまま・負は0。勾配消失を緩和、現在の主流 |
| Leaky ReLU | 負側にもわずかな傾き |
| **ソフトマックス** | 出力層で多クラス分類（合計1の確率に） |

## 学習を支える技術
- **勾配消失問題** … 層が深いと勾配が0に近づき学習が進まない → **ReLU**、**バッチ正規化**、適切な初期化で緩和。
- **最適化手法** … **SGD → Momentum → AdaGrad → RMSProp → Adam**（現在の定番）。
- **学習率** … 大きすぎ＝発散、小さすぎ＝遅い。
- **過学習対策** … **ドロップアウト**（ランダムにノードを無効化）、**早期終了**、データ拡張、正則化。
- **バッチサイズ／エポック／ミニバッチ学習**。

## 代表的なネットワーク

| 種類 | 用途 | キーワード |
|---|---|---|
| **CNN** | 画像 | **畳み込み**（特徴抽出）＋**プーリング**（位置ずれ吸収・圧縮）。LeNet/AlexNet/VGG/ResNet |
| **RNN** | 系列（言語・音声） | 過去の状態を保持。勾配消失に弱い |
| **LSTM / GRU** | 長い系列 | ゲート機構で長期依存を学習 |
| **Transformer** | 言語・生成AI | **自己注意（Self-Attention）**。並列計算可、長距離依存に強い |
| **オートエンコーダ** | 次元圧縮・異常検知 | 入力を再構成 |
| **GAN** | 生成 | 生成器 vs 識別器の敵対学習 |
| **拡散モデル** | 画像生成 | ノイズ除去を学習（Stable Diffusion等） |

### CNNの要点
- **畳み込み層**：フィルタ（カーネル）で局所特徴を抽出。
- **プーリング層**：最大値（マックスプーリング）等でダウンサンプリング、位置不変性。
- **ResNet**：**スキップ接続（残差）**で超深層でも学習可能に。

### Transformer／Attention の要点
- **Attention** … 系列のどこに注目するかを重みづけ。RNNの**逐次処理・長距離依存の弱さ**を解消し**並列化**を可能に。
- 7月のPhase1で深掘り（3Blue1Brown〜Attention）。📝「AttentionがRNNの何の問題を解決した？」

## 学習方法の発展
- **転移学習 / ファインチューニング** … 学習済みモデルを別タスクに流用。
- **事前学習＋微調整** … BERT・GPT系の基本。
- **自己教師あり学習** … ラベルなしデータから擬似タスクで学習。

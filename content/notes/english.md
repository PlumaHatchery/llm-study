# 英語学習

> 方針：**G検定本番(7/4)までは軽め（10〜15分/日）の習慣維持**。7/5から本格化。
> 中心は **① 単語・語彙**（暗記カード）＋ **② リーディング**（短い英文を毎日1本）。

## 毎日のルーティン（〜7/4：10〜15分）
1. **単語カード**「英単語」デッキを **5〜10枚**（コンテンツ → 暗記カード）。間隔反復なので、覚えたら自動で間隔が伸びる。
2. **短い英文を1本**読む（下の教材から）。分からない単語を1〜2個カードに追加。

## 本格化ルーティン（7/5〜：30〜45分）
1. 単語 15枚
2. リーディング 15分（記事1本を精読 → 知らない単語をカード化）
3. 音読 or シャドーイング 5〜10分（リスニング・発音を足したくなったら）

## おすすめ教材（無料・短い英文向き）
- **Breaking News English** — レベル別に同じニュースが用意され、易しい段階から読める
- **NHK WORLD-JAPAN / News in Slow** — 日本の話題で背景を知っているから読みやすい
- **BBC Learning English** — 語彙・文法の短い記事と音声
- **Wikipedia「Simple English」版** — 興味のある項目をやさしい英語で

> 単語は**自分のレベルに合わせて入れ替え**てOK。`data/decks/english.json` を編集して、読んでいて出会った単語を足していくのが一番伸びる。

## 単語カードの足し方
`data/decks/english.json` の `cards` に追加：
```json
{ "id": "一意のID", "front": "英単語", "back": "意味＋例文" }
```

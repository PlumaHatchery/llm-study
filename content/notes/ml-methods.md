# ② 機械学習の手法

> 計画 6/13〜6/14 の範囲。**出題ボリューム最大の本丸**。手法名と特徴をセットで。

## 学習の種類

| 種類 | 概要 | 代表タスク |
|---|---|---|
| 教師あり学習 | 入力と正解ラベルのペアで学習 | 回帰・分類 |
| 教師なし学習 | 正解なし、データの構造を発見 | クラスタリング・次元削減 |
| 強化学習 | 報酬を最大化する行動を試行錯誤で学習 | 制御・ゲーム |

## 教師あり学習の手法

- **線形回帰** … 連続値を直線で予測。**ロジスティック回帰**は分類（出力をシグモイドで0–1に）。
- **k近傍法（k-NN）** … 近いk個の多数決で分類。
- **SVM（サポートベクターマシン）** … **マージン最大化**で境界を引く。**カーネル法**で非線形分離。
- **決定木** … 条件分岐で分類・回帰。解釈しやすい。**ジニ不純度／情報利得**で分岐。
- **アンサンブル学習** … 弱学習器を組み合わせる。
  - **バギング** … 並列に学習して多数決（例：**ランダムフォレスト**）。
  - **ブースティング** … 直列に誤りを補正（例：**AdaBoost / 勾配ブースティング / XGBoost**）。

## 教師なし学習の手法

- **k-means** … クラスタ中心を更新して分割（クラスタ数kを事前指定）。
- **階層的クラスタリング** … デンドログラムで階層構造を作る。
- **主成分分析（PCA）** … 分散が最大の軸へ**次元削減**。
- **t-SNE / UMAP** … 高次元データの可視化向け次元削減。

## 強化学習の基礎
- **エージェント／環境／状態／行動／報酬**。**Q学習**、**方策（policy）**。
- **探索と活用のトレードオフ**（exploration vs exploitation）。
- DLと組み合わせた**DQN**（Deep Q-Network）→ AlphaGoへ。

## 評価指標（頻出！）

混同行列：TP / FP / FN / TN

| 指標 | 式 | 意味 |
|---|---|---|
| 正解率 Accuracy | (TP+TN)/全体 | 全体の正答割合 |
| 適合率 Precision | TP/(TP+FP) | 「陽性と判定」の正しさ（誤検知を避けたい時） |
| 再現率 Recall | TP/(TP+FN) | 「実際の陽性」の取りこぼし防止（見逃しを避けたい時） |
| F値 F1 | 2PR/(P+R) | 適合率と再現率の調和平均 |

- **トレードオフ**：適合率↑と再現率↑は両立しにくい。**ROC曲線／AUC**で総合評価。
- 例：**がん検診＝再現率重視**（見逃しNG）、**スパム判定＝適合率重視**（重要メール誤判定NG）。

## 学習の落とし穴
- **過学習（オーバーフィッティング）** … 訓練データに適合しすぎ、汎化性能が下がる。
- 対策：**正則化（L1=ラッソ / L2=リッジ）**、**交差検証**、データ増強、早期終了。
- **次元の呪い** … 次元が増えるとデータが疎になり学習困難。
- **データの分割** … 訓練／検証／テスト。**ホールドアウト**と**k分割交差検証**。

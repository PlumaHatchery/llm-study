# ② 機械学習の手法

> 計画 6/27。最大配点。**手法の帰納バイアス**と**評価指標**を正確に。

## 学習パラダイム
- **教師あり**：(x, y) から写像 f を学習。回帰（連続）／分類（離散）。
- **教師なし**：x のみ。密度推定・クラスタリング・次元削減・表現学習。
- **半教師あり / 自己教師あり**：少量ラベル＋大量未ラベル。SSLは擬似タスクで表現を獲得。
- **強化学習**：MDP上で累積報酬 E[Σγ^t r_t] を最大化する方策πを学習。探索と活用のトレードオフ。

## バイアス-バリアンス（全体を貫く視点）
汎化誤差 ≈ バイアス² + バリアンス + ノイズ。
- 高バイアス＝**未学習（underfitting）**：モデルが単純すぎ。
- 高バリアンス＝**過学習（overfitting）**：訓練に適合しすぎ、汎化が落ちる。
- 制御手段：正則化、モデル複雑度、データ量、交差検証。

## 教師あり：主要手法と帰納バイアス
- **線形/ロジスティック回帰**：線形決定境界。ロジスティックは sigmoid で対数オッズを線形モデル化。
- **SVM**：マージン最大化（構造的リスク最小化）。**ソフトマージン C** で誤分類許容を調整、**カーネルトリック**で高次元写像を内積計算のみで実現（RBF等）。
- **決定木**：軸平行分割。分岐基準は**ジニ不純度／情報利得（エントロピー減少）**。高バリアンス。
- **アンサンブル**：
  - **バギング（ランダムフォレスト）**：ブートストラップ＋特徴サブサンプリングで**バリアンス低減**、並列。
  - **ブースティング（AdaBoost / 勾配ブースティング / XGBoost）**：残差を逐次フィット、**バイアス低減**、直列。
- **k-NN**：非パラメトリック、遅延学習。次元の呪いに弱い。
- **ナイーブベイズ**：条件付き独立仮定下での事後確率最大化。

## 教師なし
- **k-means**：クラスタ内二乗和最小化（Lloyd法）。k は事前指定、初期値依存（k-means++）。
- **階層的クラスタリング**：凝集型/分割型、デンドログラム。
- **PCA**：共分散行列の固有値分解。分散最大の直交基底へ射影＝線形次元削減。
- **t-SNE / UMAP**：非線形、局所構造保存の可視化（距離は大域的に保たれない点に注意）。

## 評価指標（最頻出）
混同行列 TP/FP/FN/TN を基準に：

| 指標 | 式 | 用途 |
|---|---|---|
| Accuracy | (TP+TN)/N | 不均衡データでは誤誘導 |
| Precision | TP/(TP+FP) | FP コストが高い（スパム判定） |
| Recall(感度) | TP/(TP+FN) | FN コストが高い（がん検診） |
| F1 | 2PR/(P+R) | P と R の調和平均 |
| 特異度 | TN/(TN+FP) | 陰性の取りこぼし |

- **ROC-AUC**：閾値非依存、TPR-FPR。**PR-AUC** はクラス不均衡に頑健。
- Precision↔Recall は閾値でトレードオフ。**不均衡対策**：リサンプリング、class weight、閾値調整。

## 正則化・検証
- **L1（Lasso）**：スパース化（特徴選択）。**L2（Ridge）**：重み縮小。**Elastic Net** は両者。
- **k分割交差検証 / 層化 / Leave-One-Out**。データリーク（前処理を全データで実施等）に注意。

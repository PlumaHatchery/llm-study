# ⑤ 数理・統計

## 確率・統計の基礎
- **平均・分散・標準偏差** … ばらつきの指標。
- **正規分布** … 平均と分散で決まる釣鐘型。
- **条件付き確率** と **ベイズの定理**：P(A|B) = P(B|A)P(A) / P(B)。
  - ナイーブベイズ分類器の基礎。
- **相関と因果は別物**（相関があっても因果とは限らない）。
- **大数の法則／中心極限定理**（名前と概要だけ）。

## 線形代数（最小限）
- **ベクトル・行列・内積**。NNの計算はほぼ行列演算。
- **固有値・固有ベクトル** … PCAの背景。

## 微分・最適化
- **勾配（偏微分）** … 損失を下げる方向。**勾配降下法**の基礎。
- **連鎖律（チェインルール）** … 誤差逆伝播の数学的核心。
- **損失関数** … 回帰＝**平均二乗誤差（MSE）**、分類＝**交差エントロピー**。

## 情報量
- **エントロピー** … 不確実性の大きさ。
- **交差エントロピー／KLダイバージェンス** … 分布間の差。分類の損失に使う。

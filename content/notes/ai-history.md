# ① AIの歴史・動向

> 計画 6/12 の範囲。**人名・ブーム・冬の時代の年代**を押さえる。一番軽い章でウォームアップ。

## AIブームと冬の時代

| 時期 | 中心技術 | できたこと | 冬の原因 |
|---|---|---|---|
| 第1次（1950s–60s） | 探索・推論 | 迷路・パズル・定理証明（トイ・プロブレム） | 現実の複雑な問題が解けない |
| 第2次（1980s） | 知識表現・エキスパートシステム | 専門知識をルール化して推論 | 知識獲得のボトルネック／知識の記述・保守が困難 |
| 第3次（2000s–現在） | 機械学習・ディープラーニング | 大量データから自動で特徴を学習 | （継続中） |

- **トイ・プロブレム** … 簡単な問題は解けても現実問題に通用しない＝第1次の限界。
- **知識獲得のボトルネック** … 専門家の知識をすべて書き出すのが困難＝第2次の限界。
- **フレーム問題** … 「今やろうとしていることに関係する事柄だけ取り出す」のが難しい（ダニエル・デネット）。
- **シンボルグラウンディング問題** … 記号（シンボル）と意味を結びつけられない問題。
- **強いAI／弱いAI** … ジョン・サール。強いAI＝意識を持つ、弱いAI＝道具としての知能。**中国語の部屋**は強いAI批判の思考実験。

## 第3次ブームの転機

| 年 | 出来事 |
|---|---|
| 2006 | ヒントンらが**ディープラーニング**（オートエンコーダによる事前学習）を提唱 |
| 2012 | **ILSVRC**（画像認識コンペ）で**AlexNet**が圧勝 → DLブームの号砲 |
| 2012 | Googleの「**猫**」：大量画像から教師なしで猫の概念を獲得 |
| 2014 | **GAN**（敵対的生成ネットワーク、Goodfellow） |
| 2016 | **AlphaGo** がプロ棋士に勝利（DeepMind） |
| 2017 | **Transformer**（"Attention Is All You Need"） |
| 2022– | **ChatGPT** など大規模言語モデル（LLM）の普及 |

## 重要人物

- **アラン・チューリング** … チューリングテスト（機械が人間と区別つかなければ知能ありとみなす）。
- **ジョン・マッカーシー** … 「人工知能（AI）」という言葉を提唱（1956 **ダートマス会議**）。
- **ジェフリー・ヒントン** … DLの父。誤差逆伝播の普及、2012 AlexNet。
- **ヤン・ルカン** … CNN（畳み込み）の発展。
- **ヨシュア・ベンジオ** … 上記2名とともにディープラーニング3巨頭。

## AIのレベル分類（4段階）
1. 単純な制御プログラム（マーケティング上「AI」と称するもの）
2. 古典的な人工知能（探索・推論、知識ベース）
3. 機械学習を取り入れたAI
4. ディープラーニングを取り入れたAI
